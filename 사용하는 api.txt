2. 사용 예정 API 및 AI 정리

────────────────────────────

2-1. FoodAI (음식 이미지 인식)

목적: 사용자가 음식을 사진으로 찍으면, 이를 자동으로 인식하고 칼로리 및 영양 성분 정보를 제공합니다.

기능: 사용자가 음식을 사진으로 찍으면 FoodAI가 해당 음식의 종류를 인식하고, Nutritionix API와 연계하여 관련 영양 정보를 자동으로 제공합니다.

────────────────────────────

2-2. Nutritionix API (음식 영양 성분 제공)

목적: 사용자가 섭취한 음식의 칼로리 및 영양 성분 정보를 제공합니다.

기능: FoodAI가 인식한 음식의 이름을 Nutritionix API로 보내, 칼로리 및 영양소 정보를 자동으로 불러옵니다.

────────────────────────────

2-3. Google Fit API (운동 데이터 자동 수집)

목적: 사용자의 운동 데이터를 자동으로 수집하고 기록하여 목표 달성을 지원합니다.

기능: 사용자가 스마트폰이나 웨어러블 기기를 통해 수집한 운동 데이터를 연동하여 주간 운동 목표와 비교할 수 있도록 지원합니다.

────────────────────────────

2-4. openFDA API (약물 정보 제공)

목적: 사용자가 복용 중인 약물 정보를 검색하고 추가할 수 있도록 합니다.

기능: 사용자가 복용하는 약물에 대한 기본 정보를 제공하여 건강 상태 관리에 도움을 줍니다.

────────────────────────────
