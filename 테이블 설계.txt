1. 테이블

────────────────────────────

1-1. 회원 정보 테이블 (user_profile)

user_id: 고유 사용자 ID (기본키, 자동 증가 숫자)

login_id: 사용자가 로그인할 때 사용하는 ID

phone_number: 휴대폰 번호 (로그인 및 인증 용도)

nickname: 사용자가 설정하는 별명 (랭킹 시스템에서 익명성 보장)

name: 사용자 이름

password_hash: 비밀번호 (암호화)

temp_password: 임시 비밀번호 (계정 복구 시 사용)

temp_password_expiry: 임시 비밀번호 만료 시간

age: 나이

gender: 성별

account_deleted: 계정 삭제 여부 (Y/N)

────────────────────────────

1-2. 건강 상태 정보 테이블 (health_status)

user_id: 사용자 ID (외래키)

medication_id: 복용 중인 약물 ID (외래키, 복용 중단 시 사용)

height: 키

weight: 체중

body_fat_percentage: 체지방률 (선택 사항)

conditions: 건강 상태 (예: 당뇨, 고혈압)

activity_level: 활동 수준 (예: 주로 앉아 있음, 활동적)

last_updated: 마지막 정보 업데이트 날짜

────────────────────────────

1-3. 약물 정보 테이블 (medication_info)

medication_id: 고유 약물 ID (기본키, 자동 증가 숫자)

medication_name: 약물 이름 (검색을 위한 필드)

quantity: 사용자가 선택한 약물 개수

duration: 복용 기간 (시작일 및 종료일)

────────────────────────────

1-4. 운동 목표 설정 테이블 (exercise_goal)

goal_id: 고유 목표 ID (기본키, 자동 증가 숫자)

user_id: 사용자 ID (외래키)

exercise_type: 운동 유형 (예: 유산소 운동, 근력 운동, 유연성 및 밸런스 운동, HIIT, '기타' 선택 시 사용자 입력)

duration_per_week: 주당 목표 운동 시간 (시간/분)

frequency: 주당 운동 빈도 (회)

last_updated: 마지막 목표 업데이트 날짜
